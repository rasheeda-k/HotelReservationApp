This project is a console-based hotel booking application developed as part of my Java training. It demonstrates strong fundamentals in object-oriented programming, layered architecture, and data handling using Java Collections.

🔹 Features
Search available hotel rooms
Book and manage reservations
Customer registration and management
Admin panel for system control

🔹 Architecture
The application follows a clean modular structure:
Model Layer: Defines core entities like Customer, Room, and Reservation
Service Layer: Handles business logic and operations
API Layer: Acts as a facade between UI and services
UI Layer: Console-based interaction for users and admin

🔹 Technologies Used
Java (Core Java, OOP)
Java Collections Framework
Maven (for project structure)

🔹 Key Learnings
Applying OOP concepts in real-world applications
Designing scalable and maintainable systems
Separation of concerns using layered architecture
